## Getting started

### Download

Via npm

```
npm install fluffy-factory/magnet-mouse --save
```

or manual [download](https://github.com/fluffy-factory/magnet-mouse/archive/master.zip).

### Usage

#### ES6 modules

```javascript
import MagnetMouse from 'magnet-mouse/lib/magnet-mouse.es.js';
```
